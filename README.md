# Credit Scoring Engine-CITS2755

## Overview

The Credit Scoring Engine is a simple Java-based application that evaluates a user's creditworthiness based on financial information such as annual income, monthly expenses, loan amount, and repayment history. The system calculates a credit score and classifies the user into different credit categories.

## Features

- Calculate credit score based on financial data
- Analyze income, expenses, and loan amount
- Consider repayment history
- Categorize users as:
  - Excellent Credit
  - Good Credit
  - Average Credit
  - Poor Credit
- Simple console-based interface

## Technologies Used

- Java
- Scanner Class
- Object-Oriented Programming Concepts

## Project Structure

```
CreditScoringEngine/
```

## How It Works

1. User enters annual income.
2. User enters monthly expenses.
3. User enters loan amount.
4. User provides repayment history.
5. The system calculates a credit score.
6. The final credit rating is displayed.

## Sample Input

```
Annual Income: 600000
Monthly Expenses: 15000
Loan Amount: 200000
Repayment History: true
```

## Sample Output

```
Credit Score: 900
Excellent Credit
```

## Installation and Execution

### Compile

```bash
javac CreditScoringEngine.java
```

### Run

```bash
java CreditScoringEngine
```
